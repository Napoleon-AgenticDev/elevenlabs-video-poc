# ElevenLabs Video POC

Proof of concept experimenting with ElevenLabs API to create videos with voice and images.

## Setup

```bash
# Install dependencies
npm install

# Or with Python
pip install elevenlabs
```

## Environment Variables

Create a `.env` file:
```
ELEVENLABS_API_KEY=YOUR_API_KEY_HERE
```

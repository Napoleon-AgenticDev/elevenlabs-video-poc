🎬 Fully Autonomous

No human intervention required after setup. The pipeline generates scripts, voice, images, music, and builds final videos automatically.

🎭 Character System

Consistent AI characters with customizable appearance. ARIA features silver titanium exoskeleton with glowing cyan LED eyes.

⚙️ Configuration

Content rating (G/PG/PG13/R), music genre, camera movement - all configurable via config.yaml.

📁 Project Structure

Organized output with project folders. Each project has scenes, images, audio, and video tracked separately.

📊 MLflow Tracking

API calls, token usage, costs, and scene durations logged for production monitoring.

🔧 Flexible Scripts

SCENES dict defines story content with audio tags for dynamic voice expression.

Quick Start

# Generate video for specific episode
python3 src/autonomous_video_v3.py --project my-project

# Generate all scenes
python3 src/autonomous_video_v3.py --all --project my-project

# Generate single scene
python3 src/autonomous_video_v3.py --scene 1 --project my-project

API Integration

This pipeline integrates with:

  • ElevenLabs: Voice generation, Music generation
  • Gemini: Image generation (fallback when ElevenLabs unavailable)
  • FFmpeg: Video compilation and concatenation
